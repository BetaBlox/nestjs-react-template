# NestJs / React / Prisma Template

## Setup

- `npm install`
- `npm run docker:up`
- `npm run db:reset`

## Environment

- `cp .env.local .env`
- Modify `.env` as needed

## Development

`npm run dev`

- app: http://localhost:5173/
- api: http://localhost:3000/api

## Build

`npm run build`

## Production

`npm run build`
`npm run start`

## Test

`npm run test`
