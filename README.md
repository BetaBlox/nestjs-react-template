# Nest / React / Prisma Template

## Setup

`npm install`

`npm run docker:up`

`npm run db:reset`

## Environment

`cp .env.example .env`

Now modify `.env` as needed

## Development

`npm run dev`

- React App: http://localhost:5173/
- Nest Api: http://localhost:3000/api

## Build

`npm run build`

## Production

`npm run build`

`npm run start`

## Test

`npm run test`
